BT1

JPA (Java Persistence API) là một đặc tả trong Java EE dùng để quản lý dữ liệu quan hệ trong ứng dụng Java, cung cấp một cách chuẩn hóa để ánh xạ các đối tượng Java (POJO) với cơ sở dữ liệu quan hệ (ORM - Object-Relational Mapping). JPA định nghĩa các API và chú thích để thực hiện các thao tác như lưu trữ, truy vấn, cập nhật và xóa dữ liệu mà không cần viết câu lệnh SQL phức tạp. Hibernate là một triển khai phổ biến của JPA, cung cấp các tính năng bổ sung và tối ưu hóa để làm việc với cơ sở dữ liệu. JPA quan trọng trong phát triển ứng dụng Java vì nó giúp đơn giản hóa việc tương tác với cơ sở dữ liệu, tăng tính di động (có thể thay đổi triển khai JPA như Hibernate, EclipseLink mà không sửa đổi mã nguồn nhiều), và cải thiện hiệu suất phát triển bằng cách giảm mã lệnh thủ công. Nhờ JPA, lập trình viên có thể tập trung vào logic nghiệp vụ thay vì quản lý cơ sở dữ liệu.
